# solitairegame

This Java program is based on the Peg Solitaire game

Peg Solitaire is a single-player board game where the player jumps pegs/marbles over adjacent ones, removing
the jumped pegs/marbles. The game starts with filling the entire board with pegs, but the central hole
remains empty. The player then makes valid moves and aims to empty the entire board except for a solitary
peg. A valid move for a player is to jump a peg orthogonally or diagonally over an adjacent peg into a hole
two positions away. The player then removes the jumped peg.

## Board Size
Allows you to adjust the size of the board you are going to play on with a value spinner that contains the values (5,7,9,11)

## Board Type
Allows you to change the board style you are going to play on with radio buttons that have the option of (English,Hexagon,Diamond)

## Randomize Button
When pressed, randomly changes the state of the pegs on the board a random amount of times.

## Automatic vs Manual
Radio buttons that change whether the game is played by a human player (Manual) or a completely random algorithm (Automatic)

## Replay
The inputs made in the game are recorded whether or not it is Automatic or Manual. After a game has been completed the user can press the Replay button to rewatch the game that was just played.  

## Replay Slider
Adjusts the speed at which the replay is played at. CURRENTLY NON FUNCTIONAL
